This app contains a file called "./app/src/main/res/values/API_KEY.xml" that is ignored by Git.

This file contains the API key to the "http://api.football-data.org" website service.  In order
for this application to work you will need to create this file and populate with the API key
given to you.

The format of the file is as contained within the lines:

----------------------------------
<?xml version="1.0" encoding="utf-8"?>
<resources>
    <string name="API_FOOTBALL_DATA_ORG">xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx</string>
</resources>
----------------------------------
